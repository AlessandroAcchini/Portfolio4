Misfeets Tournoi

Plateforme compétitive complète pour gérer des tournois League of Legends. Le site permet l'inscription d'équipes, la synchronisation automatique des rangs via l'API Riot Games, et la génération de statistiques en temps réel.

Voir le site

Fonctionnalités

  • •Inscription rapide des équipes
  • •Synchronisation automatique des rangs via l'API Riot
  • •Génération de statistiques en temps réel
  • •Système d'équilibrage automatique des équipes
  • •Lien OP.GG Multisearch généré automatiquement
  • •Cash prize pour les vainqueurs

Technologies

PHP MySQL HTML / CSS JavaScript API Riot Games
